Module 00 / Prerequisite

投資手法の全体マップ

選ぶ前に、全体像を知る。
9つのアプローチの構造・エッジ・弱点を俯瞰し、「どこで戦うか」を自分で決める。

Premise

なぜ全体マップが必要か

投資手法は1つではありません。それぞれのアプローチは、異なる前提・異なる市場環境で利益を生む構造を持っています。1つの手法しか知らない状態でトレードすると、その手法が機能しない環境に入ったとき、原因の切り分けができない。

「知ったうえで選ぶ」のと「それしか知らない」のは違う。全体マップを持つことで、自分が選んだ手法の位置づけを理解し、トレードジャーナルの分析にも深みが出ます。うまくいかないとき、「手法自体の問題」なのか「市場環境との不一致」なのか「自分の実行の問題」なのかを区別できるようになる。

すべての戦略を知る必要はない。しかし、自分の戦略が「何に賭けていて、何を捨てているのか」は知らなければならない。

Framework

投資手法を分類する2つの軸

すべての投資手法は、大きく2つの軸で整理できます。

軸1：何に賭けるか（リターンの源泉）
モメンタム（上がっているものはさらに上がる）か、ミーンリバージョン（行き過ぎたものは戻る）か、バリュー（本来の価値より安いものは是正される）か、イベント（特定の出来事が価格を動かす）か。

軸2：どう判断するか（分析の方法）
テクニカル（チャート・出来高・価格パターン）か、ファンダメンタル（業績・財務諸表・経済指標）か、クオンツ（統計モデル・アルゴリズム）か。

同じ「モメンタム」でも、テクニカルで解釈すれば移動平均線のクロス、ファンダメンタルで解釈すればEPSの加速、クオンツで解釈すれば過去6ヶ月リターンのファクタースコアになる。同じエッジを違う言語で記述しているだけというケースも多い。

The Nine Approaches

9つのアプローチ

各アプローチをクリックして展開してください。構造・エッジ・弱点・あなたのCPA背景との相性を評価しています。

📈

① トレンドフォロー / モメンタム

上がっている株を買い、トレンドに乗る
+

リターンの源泉

  • 株価のモメンタム（慣性）。上昇中の株はさらに上昇しやすいという統計的事実に賭ける
  • 機関投資家の大口買いが数週間〜数ヶ月かけて株価を押し上げるプロセスに「相乗り」する構造
  • 学術的にも「モメンタムファクター」として広く認知されている市場アノマリー
時間軸
数日〜数ヶ月
勝率
35〜50%
利益構造
損小利大（低勝率・高R倍数）
代表的手法
SEPA, CAN SLIM, Turtle

強い環境 / 弱い環境

  • 強い：明確な上昇トレンド、セクターローテーションが活発な相場
  • 弱い：レンジ相場（横ばい）、急激なトレンド反転。モメンタムクラッシュが最大のリスク

代表人物

  • Weinstein, Minervini, O'Neil, Richard Dennis（タートルズ）
🔶 CPA背景との相性：中程度

テクニカル主体のため会計知識の直接的な優位性は小さい。ただしCAN SLIMのファンダ部分（EPS分析・決算の質の評価）では活きる。Module 1〜8で体系化済み。

💎

② バリュー投資

本来の価値より安い株を買い、市場が気づくのを待つ
+

リターンの源泉

  • 株価が「内在的価値（intrinsic value）」を下回っているときに買い、価格が価値に収束するプロセスで利益を得る
  • 市場が短期的に過度に悲観的になる瞬間を狙う。恐怖で売られた株を拾う
  • 「安全域（margin of safety）」——十分に安く買えば、分析が多少間違っていても損をしにくい
時間軸
数ヶ月〜数年
勝率
比較的高い（忍耐が前提）
利益構造
中勝率・中〜高R倍数
代表的手法
DCF, PER/PBR分析, GARP

強い環境 / 弱い環境

  • 強い：暴落後の回復局面、不人気セクターの見直し
  • 弱い：バブル相場（割安株が永遠に無視される）、「バリュートラップ」（安いが構造的に衰退する企業）

代表人物

  • Benjamin Graham, Warren Buffett, Seth Klarman, Peter Lynch（GARP）
🟢 CPA背景との相性：非常に高い

財務諸表の読み解き、会計の質の評価（粉飾や利益操作の兆候発見）、税効果会計・退職給付会計の裏側の理解。これらはバリュー投資家にとって最も重要なスキルであり、CPAは構造的に有利。ただし時間軸が長く、週次スイングとは相性が悪い。

🔄

③ ミーンリバージョン（平均回帰）

行き過ぎた動きは元に戻るという統計に賭ける
+

リターンの源泉

  • 株価が短期的に平均から大きく乖離した後、平均に回帰する傾向を利用する
  • RSIが30以下（売られすぎ）で買い、70以上（買われすぎ）で売る、といった逆張り戦略
  • ボリンジャーバンドの−2σで買い、+2σで売るなどの統計的アプローチ
時間軸
数日〜数週間
勝率
比較的高い（60〜70%）
利益構造
高勝率・低R倍数（小さく多く勝つ）
代表的手法
RSI逆張り, ボリンジャー, ペアトレード

強い環境 / 弱い環境

  • 強い：レンジ相場、ボラティリティが安定している環境
  • 弱い：強いトレンド相場。売られすぎの株がさらに売られ続ける「落ちるナイフ」リスク
⚪ CPA背景との相性：低い

純粋に統計・テクニカル的なアプローチであり、会計知識の優位性はほぼ活かせない。ただしモメンタム戦略との組み合わせ（モメンタムが機能しないレンジ相場で切り替える）として知っておく価値はある。

⚡

④ イベントドリブン

決算・M&A・税制改正など特定のカタリストに賭ける
+

リターンの源泉

  • 特定のイベント（決算発表、業績修正、M&A、スピンオフ、規制変更）が株価を動かすプロセスに賭ける
  • 「イベント前にポジションを取り、イベント後の値動きで利確する」または「イベント直後の過剰反応の修正を狙う」
  • 決算サプライズ後の株価ドリフト（Post-Earnings Announcement Drift）は学術的にも認知されたアノマリー
時間軸
数日〜数週間
勝率
イベントの質に依存
利益構造
イベントごとに非対称リスク設計
代表的手法
決算プレイ, スピンオフ, 特殊状況

強い環境 / 弱い環境

  • 強い：あらゆる市場環境で機会がある（イベントはトレンドに関係なく発生する）
  • 弱い：イベントが「織り込み済み」の場合、動かない。決算ギャンブルは損切りが効かないリスク（GD）
🟢 CPA背景との相性：極めて高い

決算短信を他の参加者より正確に速く読める、業績修正の可能性を進捗率から予測できる、税制改正のインパクトを構造的に理解できる。この3つはCPAだけが持つ圧倒的なエッジ。さらに時間軸も週次と合致。最も探求する価値のある領域。

🔢

⑤ クオンツ / 統計的手法

データとアルゴリズムで再現可能なパターンを見つける
+

リターンの源泉

  • 過去のデータからバックテストで検証された統計的パターンに賭ける
  • ファクター投資（モメンタム・バリュー・クオリティ・低ボラティリティなどの「ファクター」で銘柄を選ぶ）
  • 人間の判断バイアスを排除し、ルールベースで完全に機械的に売買する
時間軸
様々（秒〜月）
勝率
戦略に依存
利益構造
統計的エッジの積み重ね
代表的手法
ファクター投資, 統計的裁定

強い環境 / 弱い環境

  • 強い：十分な流動性とデータがある市場。パターンが安定している期間
  • 弱い：レジーム変化（市場の構造が変わるとき）。過学習（overfitting）で見かけのパターンに騙される
🔶 CPA背景との相性：中程度

Python/GASのスキルがあるので技術的には可能。ただし63万円の資本規模ではクオンツのメリット（分散・頻度）が活きにくい。将来的に資本が拡大した段階で検討する価値あり。

🏗

⑥ Wyckoff / マーケット構造分析

機関投資家の「足跡」をチャートから読む
+

リターンの源泉

  • 機関投資家（"コンポジットマン"）がどこで買い集め（accumulation）、どこで売り抜け（distribution）しているかをチャートから識別する
  • Weinsteinの4ステージとルーツを共有。Stage 1のaccumulationフェーズをより精密に分析する手法
  • 出来高と価格の関係（effort vs result）で需給バランスを判断する
時間軸
数日〜数ヶ月
習得難度
高い（主観的判断が多い）
代表的手法
Wyckoff法, VSA

強い環境 / 弱い環境

  • 強い：出来高が透明に可視化される市場。流動性のある銘柄
  • 弱い：ダークプール取引が多い環境。小型株で出来高データが不十分な場合
⚪ CPA背景との相性：低い

純粋にチャートリーディングの技術。会計知識との接点は少ない。ただしStage Analysisとの連続性があり、Module 1の理解を深める補助教材としては有用。

☁

⑦ 一目均衡表

日本発の総合テクニカルシステム
+

リターンの源泉

  • 転換線・基準線・先行スパン（雲）・遅行スパンの5要素で、トレンド方向・強さ・サポート/レジスタンスを1つの画面で判断
  • 「時間論」「波動論」「値幅観測論」の3つの理論体系を持つ、非常に奥が深いシステム
  • 日本の個人投資家に広く使われているため、多くの参加者が同じ水準を意識している＝自己実現的に機能しやすい
時間軸
数日〜数ヶ月
習得難度
高い（奥が極めて深い）
日本市場との親和性
非常に高い

強い環境 / 弱い環境

  • 強い：トレンドが明確な相場。日本株では特に機能しやすい
  • 弱い：レンジ相場（雲の中で株価がもみ合うと判断不能になる）
⚪ CPA背景との相性：低い

純粋なテクニカルシステム。ただし日本市場で戦うなら「多くの参加者が見ている指標」として知っておく価値はある。雲がサポートやレジスタンスとして機能する場面は実際に多い。

💰

⑧ インカム / 配当戦略

配当収入で安定的なリターンを積み上げる
+

リターンの源泉

  • 高配当利回りの銘柄を保有し、定期的な配当収入を得る
  • 配当の再投資による複利効果で長期的に資産を成長させる
  • 株価の上下よりもキャッシュフローの安定性を重視
時間軸
数年〜永久
利益構造
低リスク・低リターン（安定的）

週次スイングとの関係

  • スイングトレードとは対極の手法。63万円で短期の能力構築を目指す今のフェーズでは適用外
  • ただし、将来的にトレード資金とは別に「守りの資金」をどこに置くかの選択肢として知っておく価値はある
🔶 CPA背景との相性：中程度

配当の持続可能性を財務分析で判断する能力はCPAの強み。ただし今のフェーズでは使わない。

🌍

⑨ マクロ / グローバル

経済サイクル・金利・地政学でポジションを取る
+

リターンの源泉

  • 景気サイクル、金利動向、為替変動、地政学的リスクなど、マクロ経済の大きな変動に賭ける
  • 「この政策が実施されれば、このセクターが恩恵を受ける」という因果推論
  • 個別株ではなく、ETF・先物・通貨・債券も含む広い視野
時間軸
数ヶ月〜数年
必要知識
経済学、中央銀行政策、地政学

強い環境 / 弱い環境

  • 強い：大きな経済転換点（利上げサイクル開始、景気後退入り等）
  • 弱い：マクロテーゼが正しくても、タイミングが早すぎると損切りに追い込まれる
🔶 CPA背景との相性：中程度

税制改正の波及分析はマクロ→セクターの因果推論に近い。Module 7のセクターローテーションと接続する部分がある。ただし63万円で個別株スイングを行う今のフェーズでは、マクロは「市場全体の方向性判定（M条件）」に使う程度。

Key Insight

この地図から見えること

CPA背景との相性で突出しているのは④イベントドリブンと②バリュー投資です。どちらも財務諸表の深い読解が直接的なエッジになる。

ただしバリュー投資は時間軸が長く、週次スイングとは合わない。イベントドリブンは時間軸も合致し、決算サプライズ・業績修正・税制改正という「CPAが最も得意なイベント」で戦える構造。

一方、①トレンドフォロー（Module 1〜8）はCPA背景のエッジは中程度だが、ルールが明確で学習しやすく、63万円の「授業料」で体系的にスキルを身につけるには最適。

💡 最も有力な組み合わせ：①トレンドフォローを基盤として習得しつつ（Module 1〜8）、④イベントドリブンの要素を重ねる。具体的には、トレンドテンプレートを通過した銘柄の中から、決算サプライズや業績修正が期待できる銘柄を優先的に選ぶ。スクリーニングはトレンドフォロー、銘柄の最終選定はCPAの決算分析眼。2つのエッジを重ねることで、単独の手法より高い期待値を狙える可能性がある。

ただし、これは現時点での仮説にすぎません。実際にトレードデータが溜まった段階で、どのアプローチが自分に合っているかを検証すべきです。全体マップを持っていれば、「この負けはモメンタムが機能しない環境だったのか」「イベントドリブン的に拾った銘柄のほうが成績がいいのか」といった分析ができる。それがこのModule 0の存在意義です。

Navigation

この全体マップの使い方

1
まずModule 1〜8を一通り理解する。トレンドフォローは最も体系化されていて、学習の骨格として優れている。
2
実戦に入ったら、ジャーナルに「この手法のどの要素が機能したか」を記録する。トレンドテンプレートの通過が効いたのか、決算分析が効いたのか。
3
20〜30トレード後に、自分のエッジがどこにあるかを分析する。ファンダ分析で選んだ銘柄の成績が良ければ、イベントドリブンの比重を上げる。テクニカルだけで十分なら、そのまま継続。
4
市場環境が変わったとき、この地図に戻る。トレンドフォローが機能しない時期に、ミーンリバージョンやイベントドリブンに切り替える判断材料として使う。
