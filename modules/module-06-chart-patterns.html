Base Patterns

ベースパターン——すべてのブレイクアウトの母体

Module 2でVCPを学びましたが、VCPはブレイクアウトパターンの一つに過ぎません。O'Neilの研究では、大幅上昇する銘柄はブレイクアウト前に必ず「ベース」（横ばいの調整期間）を形成します。そのベースの形状には複数の類型があります。

重要なのは、すべてのパターンに共通する原則はModule 1〜5で学んだことと同じだということ。Stage 2にある銘柄が、調整期間を経て、出来高を伴ってレジスタンスを突破する。パターンの形は違っても、このメカニズムは共通です。

ここではスイングトレードで最も使える4つのパターンを学びます。

Pattern 01

カップ・ウィズ・ハンドル

O'Neilが最も有名にしたパターン。上昇後のU字型の調整（カップ）と、小さな下方への揺れ（ハンドル）で構成されます。

詳細と注意点

有効なカップと無効なカップの見分け方
+

有効なカップの条件

  • 形状がU字型であること。V字型の急落→急回復は有効なベースではない（売り手が整理されていない）
  • 先行する上昇トレンドが存在すること（Stage 2であること）
  • カップの底部で出来高が減少していること（売り圧力の枯渇）
  • 左の淵と右の淵の価格がおおむね同じであること

ハンドルの条件

  • ハンドルはカップの上半分に形成されること。下半分に垂れ下がるハンドルはディストリビューション（売り抜け）の兆候
  • ハンドル期間中の出来高は減少すること
  • ハンドルの下落幅はカップの深さの10〜15%以内

VCPとの関係

  • VCPは「カップ・ウィズ・ハンドルの一般化」とも言える。カップが1回の大きな収縮、ハンドルが最後の小さな収縮
  • VCPのほうが収縮の回数が多く（2〜4回）、より段階的にボラティリティが縮小する
  • どちらも同じ原則：売り手の枯渇 → タイトな収束 → 出来高を伴うブレイクアウト

Pattern 03

フラットベース

強い上昇の後、株価がほとんど下がらず横ばいで推移するパターン。調整幅が非常に小さい（10〜15%以内）のが特徴で、売り手がほぼいないことを示します。

💡 フラットベースの意味：大きく上がった株がほとんど下がらない＝売りたい人がいない。これは機関投資家が利確せずにホールドし続けている証拠で、次の上昇への燃料がまだ残っていることを意味します。4つのパターンの中で最も強い銘柄に出やすい。

Volume Analysis

出来高分析——パターンの確信度を測る

すべてのベースパターンに共通する出来高のルールを体系化します。

1
ベース形成中は出来高が減少する。売り手が枯渇していくプロセスが、出来高の減少として可視化される。出来高が減らないベースは、まだ売り圧力が残っている。
2
ブレイクアウト日は出来高が平均の40〜50%以上増加する。これが機関投資家の参入シグナル。出来高が平均以下のブレイクアウトは、60〜70%の確率で失敗する。
3
上昇日に出来高が大きく、下落日に出来高が小さいパターンを探す。これはアキュミュレーション（機関投資家の買い集め）のサイン。逆に、下落日に出来高が大きいのはディストリビューション（売り抜け）。
4
ブレイクアウト後の最初のpullbackで出来高が減少することを確認する。pullback時に出来高が増加している場合、ブレイクアウトが失敗する可能性がある。健全なpullbackは「静かな戻り」。

⚠ 出来高のない動きは嘘をつく。株価だけ見てパターンを判断しない。出来高が確認しないブレイクアウトは、どんなに形がきれいでもエントリーしない。

False Breakouts

フェイクブレイクアウトの識別

ブレイクアウトの全部が成功するわけではありません。フェイク（偽）ブレイクアウトを識別するためのチェックポイント：

✕
出来高が伴わない。最も信頼性の高いフィルター。出来高が平均以下ならエントリーしない。
✕
市場全体が弱い（M条件）。日経平均/TOPIXがStage 4やディストリビューション・デイが溜まっている環境では、個別のブレイクアウトも失敗しやすい。
✕
ベースが「ルーズ」。ベース内の値動きが荒く、タイトに収束していない。VCPの収縮が段階的に小さくなっていない。
✕
ブレイクアウト初日に長い上ヒゲ。日中に大きく上げてから引けにかけて売られた＝供給がまだ残っている。
✕
トレンドテンプレートを満たしていない。パターンの形だけで判断し、Module 2の8項目チェックを省略している。

💡 フェイクブレイクアウトへの対処：損切りが設定されていれば、フェイクブレイクアウトは「小さな損失」で済む。問題は損切りを設定していない場合。パターンが失敗しても、ルール通りに対処すれば1Rの損失で次に進める。
