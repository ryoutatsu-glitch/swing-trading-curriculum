Why This Comes First

なぜStage Analysisが最初なのか

1962年、Stan Weinsteinはファンダメンタルズ的に優良とされた銘柄を保有していたにもかかわらず、市場暴落で大きな損失を被りました。この経験が彼の研究の出発点になります。

その後の数十年にわたる研究で、彼はひとつの発見に到達しました。すべての株は、下落も上昇も含めて、4つのステージを循環的に繰り返している。そしてこの構造を見れば、「いつ買うべきか」「いつ手を出してはいけないか」が事前にわかる。

Mark Minerviniのトレンドテンプレート（Module 2で学ぶ）は、このStage Analysisの上に構築されています。Weinsteinの4ステージを理解せずにMinerviniを学んでも、ルールの「なぜ」がわからないまま表面的にコピーするだけになります。だから最初に学びます。

30週移動平均線の方向があなたのトレードアイデアと一致しないなら、あなたのアイデアがおそらく間違っている。

— Stan Weinstein
The Four Stages

4つのステージ——株価のライフサイクル

すべての株は以下の4つのステージを繰り返します。各ステージは、株価の構造・30週移動平均線の方向・出来高のパターンの3要素で識別します。

白実線 = 株価　紫破線 = 30週移動平均線　下部バー = 出来高
The Compass

30週移動平均線——唯一のコンパス

Weinsteinの手法は極めてシンプルです。使うテクニカル指標は基本的に30週移動平均線（30-week SMA）ひとつ。日足チャートでは約150日移動平均線に相当し、近年Weinstein自身は200日移動平均線も同等に使えると述べています。

この移動平均線の方向が、現在のステージを教えてくれます。

判定は3要素のセットで行います。

① 株価と30週MAの位置関係（上か下か）
② 30週MAの傾き（上向き・横ばい・下向き）
③ 出来高の変化（増加・減少・急増）

この3つが揃ったとき、ステージの判定に確信が持てます。逆に、3つのうち1つでも矛盾していれば、ステージの遷移途中（まだ確定していない）可能性があります。

Deep Dive

各ステージの詳細

各ステージをクリックして展開してください。識別方法・特徴・取るべきアクションを解説します。

S1

底固め — Basing / Accumulation

下落の終わり。静寂の中で機関投資家が仕込む
+

識別シグナル

  • 長い下落トレンドの後、株価が横ばいで推移し始める
  • 30週移動平均線がフラット化する（下向きから水平へ）
  • 株価がMAの上下を行き来する（方向感がない）
  • 出来高は全体的に低下。市場の関心が薄い
  • 値動きは不規則で、レンジ内での上下動

なぜこうなるのか

  • 売り手が出尽くし、下落圧力が消えている
  • 機関投資家がこの段階で静かに買い集め（accumulation）を始める
  • 一般投資家はまだ恐怖心が強く、この銘柄に興味を持たない

このステージでやるべきこと

✕ やらないこと

買わない。「安いから」「底値に見えるから」という理由でエントリーしない。Stage 1の株は何ヶ月も横ばいを続ける可能性があり、資金が拘束される。

○ やること

ウォッチリストに入れて監視する。Stage 2への遷移シグナル（出来高急増を伴うレジスタンス突破）を待つ。

S2

上昇 — Advancing / Markup

唯一の買いゾーン。大きなリターンはここで生まれる
+

識別シグナル

  • 株価がStage 1のレジスタンス（上値抵抗線）を出来高の急増を伴って上抜ける
  • 30週移動平均線が上向きに転換する
  • 株価が30週MAの上に位置し、MAがサポートとして機能する
  • 調整（pullback）があっても30週MAで反発する
  • 出来高は上昇日に大きく、下落日に小さい

エントリーポイント

  • ベストタイミング：Stage 1のレジスタンスを出来高2〜3倍で突破した直後
  • セカンドチャンス：ブレイクアウト後の最初のpullback（30週MAまでの押し目）
  • いずれの場合も、30週MAが上向きであることが前提条件

このステージでやるべきこと

○ やること

ここで買う。Stage 2が確認できた銘柄だけがエントリー候補。ポジションを持ち、30週MAが上向きである限りホールドする。pullbackでの追加も検討できる。

○ ストップロスの設定

エントリー価格から−5%〜−8%、またはブレイクアウトしたレジスタンスの直下に損切りラインを置く。これは「Stage 2の判断が誤りだった場合」の保険。

S3

天井形成 — Distribution / Top

上昇の終わり。機関投資家が売り始める
+

識別シグナル

  • 長い上昇の後、株価が横ばいに転じる
  • 30週移動平均線がフラット化する（上向きから水平へ）
  • 株価がMAの上下を頻繁に行き来する（方向感を喪失）
  • 出来高が下落日に大きくなる——これが最も重要な警告サイン
  • 値動きの振幅が大きく不安定になる

なぜこうなるのか

  • Stage 2で大きく利益を得た機関投資家が、保有株を徐々に売却（distribution）している
  • メディアや一般投資家はまだ強気で、機関の売りを吸収している
  • 需給バランスが崩れつつあるが、表面上はまだ「高値圏」にいるように見える

このステージでやるべきこと

✕ やらないこと

新規で買わない。「最高値付近だから上がるだろう」は危険な思考。Stage 3の株は次にStage 4（下落）に移行する可能性が高い。

○ やること

Stage 2から保有していた場合、利確または部分利確のタイミング。30週MAを明確に下回ったら全ポジションを手仕舞いする。

S4

下落 — Declining / Markdown

絶対に買ってはいけないステージ
+

識別シグナル

  • 株価がStage 3のサポートを下抜ける
  • 30週移動平均線が明確に下向きに転換
  • 株価は30週MAの下に位置し、MAがレジスタンス（上値抵抗）として機能
  • 反発があっても30週MAで跳ね返される
  • 出来高は大きな下落日に膨らむ（パニック売り）

このステージでやるべきこと

✕ 絶対にやらないこと

Stage 4の株を買わない。「割安だから」「下がりすぎだから」「そろそろ反発するだろう」——すべて間違い。Stage 4の株は新しいStage 1が形成されるまで下がり続ける可能性がある。ナンピン（買い下がり）は最悪の行動。

○ やること

Stage 2から保有していた場合、まだ持っていたなら即座に手仕舞い。Stage 1への遷移が確認されるまで完全に無視する。資金は他のStage 2銘柄に回す。

Confirmation Signal

出来高——ステージ遷移の確認装置

株価の動きだけではステージの遷移は確定しません。出来高が「確認」として機能します。

↑

ブレイクアウト時の出来高

Stage 1 → Stage 2 の遷移
  • レジスタンスを突破する日の出来高が平均の2〜3倍以上であること
  • 出来高が伴わないブレイクアウトは「フェイク」の可能性が高い
  • 出来高の急増 = 機関投資家が本格的に参入しているサイン
⚠

天井圏の出来高パターン

Stage 2 → Stage 3 の遷移警告
  • 下落日に出来高が増加し始める——最も重要な警告
  • 上昇日は出来高が減少——買い手の力が弱まっている
  • この非対称パターンが数週間続いたら、Stage 3入りを疑う
Iron Rules

Stage Analysisの鉄則

ルールは少なく、しかし絶対に破らない。Weinsteinの手法の強さはこのシンプルさにあります。

1
買っていいのはStage 2の銘柄だけ。Stage 1は「まだ」、Stage 3は「もう遅い」、Stage 4は「論外」。この規律を守るだけで、大半の致命的な損失を回避できる。
2
30週移動平均線の方向に逆らわない。MAが下向きの銘柄を買ってはいけない。MAが上向きの銘柄を空売りしてはいけない。
3
ブレイクアウトには出来高の確認を求める。出来高が伴わない上抜けは信用しない。平均の2倍以上が基準。
4
市場全体のステージも見る。日経平均やTOPIXがStage 4にあるとき、個別銘柄でStage 2を買っても逆風の中で戦うことになる。地合いが悪いときは現金で待つ。
5
Stage 3の兆候が出たら、利確を躊躇しない。「もう少し上がるかも」はStage 3で最も危険な思考。出来高パターンの変化を見逃さない。
