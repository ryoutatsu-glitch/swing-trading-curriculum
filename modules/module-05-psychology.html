The Discipline Gap

知識と実行の断裂

Module 1〜4で学んだルールは、どれも理解が難しいものではありません。Stage 2でだけ買う。VCPのブレイクアウトで入る。損切りは−5〜8%。リスクは口座の2%まで。

しかし、実際にお金がかかった瞬間、これらのルールを守れなくなる。これは意志力の問題ではなく、人間の脳の構造上の問題です。

KahnemanとTverskyが1979年に発表したプロスペクト理論は、この「合理的に考えればわかるのに、実際にはそう行動できない」パターンを数学的に記述しました。トレーディングにおけるほぼすべての心理的な失敗は、この理論で説明できます。

Prospect Theory

プロスペクト理論——損失の痛みは利益の喜びの2倍

プロスペクト理論の核心は3つの発見です。

1

損失回避（Loss Aversion）

同じ金額でも、失う痛みは得る喜びの約2.25倍
+

メカニズム

  • 1万円を得たときの喜びを「1」とすると、1万円を失ったときの痛みは「2.25」
  • だから人は損失を確定することを極端に嫌う。含み損のポジションを持ち続ける
  • 逆に、利益が出ているポジションはすぐに確定したくなる（「消えてしまう前に確保しよう」）

トレードへの影響

  • 損切りができない——損失を確定させる行為は脳にとって物理的な痛みに近い。だから「もう少し待てば戻る」と自分を騙す
  • 利確が早すぎる——利益が出たら、それが消える恐怖のほうが、さらに伸びる期待より強くなる
  • 結果として「損大利小」——Module 4で学んだ期待値がマイナスの構造を、自らの心理で作り出してしまう
2

参照点依存（Reference Dependence）

絶対値ではなく、「基準点からの変化」で判断する
+

メカニズム

  • 人間は結果を絶対的な資産額ではなく、購入価格（参照点）からの変化として評価する
  • 1,500円で買った株が1,600円のとき、あなたは「資産が16万円ある」と考えるのではなく、「1万円得をしている」と考える
  • この参照点は合理的な投資判断とは無関係。株は「あなたがいくらで買ったか」を知らない

トレードへの影響

  • 「買値に戻ったら売ろう」——損失回避と参照点依存の合わせ技。買値に意味はないのに、そこが心理的なゴールになる
  • ナンピン（買い下がり）——参照点（平均取得価格）を下げることで、心理的に「損失を小さく見せる」行為。実際にはリスクが倍増している
3

ディスポジション効果（Disposition Effect）

勝っている株を売り、負けている株を持ち続ける
+

メカニズム

  • プロスペクト理論の直接的な帰結。利益領域ではリスク回避（早く確定したい）、損失領域ではリスク追求（逆転を待ちたい）
  • 1985年にShefrinとStatmanが命名。以降、世界中のマーケットで個人投資家に普遍的に観察されている
  • プロのトレーダーはこの効果が著しく小さい——つまりこれは克服可能な偏りである

Module 1〜4との矛盾

  • ディスポジション効果に従うと、Stage 2で上昇中の株を利確し、Stage 4で下落中の株を保有し続ける——Module 1のルールの正反対の行動を取ることになる
  • Module 4の「損小利大」と完全に逆。勝ちを小さく、負けを大きくする構造
  • だからこそ機械的な損切りルールが必要になる。ルールは、この心理的偏りを上書きするためのものである
脳がやりたがること

利益が出たら→すぐ確定したい
損が出たら→持ち続けたい
連敗したら→サイズを上げて取り返したい

VS
ルールが求めること

利益が出たら→トレーリングストップで伸ばす
損が出たら→−5%で即座に切る
連敗したら→サイズを半減する

Cognitive Biases

トレードを歪める認知バイアス

プロスペクト理論以外にも、トレードの判断を歪める認知バイアスがあります。

⚡

確証バイアス（Confirmation Bias）

自分に都合のいい情報だけ見えてしまう
+
  • ポジションを持った瞬間、その銘柄の良いニュースは目に入り、悪いニュースは無視する
  • 「この株が上がる理由」を探し、「上がらない理由」を見ない
  • 対策：エントリー前に「この株を買わない理由」を3つ書き出す。それでも買うなら、理由は強いということ
⚡

アンカリング（Anchoring）

最初に見た数字に判断が引きずられる
+
  • 「先週2,000円だったのに今1,500円だから安い」——先週の価格に判断がアンカーされている
  • 株価の「安い・高い」はチャートの過去の値ではなく、トレンドテンプレートやVCPの条件で判断すべき
  • 対策：常に「今この価格でゼロからポジションを建てるか？」と自問する
⚡

直近偏重（Recency Bias）

直近の結果に過度に引きずられる
+
  • 3連勝すると「自分は上手い、もっとサイズを上げよう」と思う
  • 3連敗すると「この手法は駄目だ、別の方法を試そう」と思う
  • どちらもサンプル数が足りない段階での判断。Module 4で学んだ期待値は、最低20〜30トレードのデータが必要
  • 対策：結果ではなくプロセスを評価する。「ルール通りにトレードできたか」が基準
⚡

サンクコスト（Sunk Cost Fallacy）

「ここまで待ったのだから」で判断を歪める
+
  • 含み損のポジションを持ち続ける理由が「もう3週間持っているから」になる
  • 過去に費やした時間や感情的投資は、今後の正しい判断には一切関係がない
  • 対策：「今このポジションを持っていなかったとして、今日この価格で新規に買うか？」答えがNoなら売る
Tilt

ティルト——感情がシステムを上書きする瞬間

ゲーム理論やTFTの文脈では馴染み深い概念でしょう。ティルトとは、感情的な反応がルールベースの判断を上書きする状態です。トレードにおいてティルトは最大の破壊要因です。

!

ティルトの典型パターン

認識できれば対処できる
+

リベンジトレード

  • 損切り直後に「取り返してやる」と衝動的にエントリー。VCPもトレンドテンプレートも無視
  • 感情の源泉は損失回避バイアス。「今日をプラスで終わりたい」という参照点依存
  • TFTで言えば、負けた直後に「取り返す」ためにハイリスクな構成を組むのと同じ。勝率は下がる

オーバートレード

  • 連勝後に「自分は特別だ」と感じ、ポジションサイズを拡大。2%ルールを破る
  • 直近偏重バイアス＋自信過剰（Overconfidence）の合わせ技
  • 相場が変化して連敗したとき、拡大したサイズが致命的なドローダウンを生む

分析麻痺（Analysis Paralysis）

  • 連敗の恐怖から、条件を満たす銘柄が目の前にあるのにエントリーできない
  • 「もっと確実なセットアップを待とう」が無限に続く。機会損失が積み上がる
  • これもティルトの一形態。恐怖がルールを上書きしている

💡 ティルトの検出法：「いつもと違う判断をしようとしている自分」に気づくこと。ルールに書いていないことをやろうとしている瞬間、それはティルト。トレードする前に「これはルールに基づいた判断か、感情に基づいた判断か」を1秒だけ自問する。

Countermeasures

心理バイアスへの実践的な対策

バイアスは「知っている」だけでは消えません。Kahnemanも認めているように、自分のバイアスを知っていても、それを経験する強さは変わらない。だからこそ仕組みで対処する必要があります。

1
エントリー時に損切り注文を同時に入れる。「後で考える」は100%感情に負ける。逆指値注文として機械的にシステムに入れる。判断の余地を残さない。
2
トレード前にジャーナルに仮説を書く。「なぜ買うのか」「何が起きたら売るのか」を言語化してから発注する。書けないなら、それはトレードすべきではないというシグナル。
3
連敗後は1日休む。3連敗したら、翌日はチャートを見ない。休むこと自体がルール。「何もしない」もトレードの一部。
4
週次レビューでプロセスを評価する。「今週いくら稼いだか」ではなく「今週ルールを何回守れたか」を数える。ルール遵守率80%以上を目標にする。利益は結果。プロセスが原因。
5
トレードの結果と判断の質を分離する。ルール通りにトレードして損切りになった → 良いトレード。ルールを破って偶然利益が出た → 悪いトレード。長期的には良いトレードを積み重ねたほうが勝つ。

最も優れたトレーダーとは、最も多く正しい予測をする人ではない。間違ったときに最も速く損失を限定できる人だ。

— Mark Minervini
The Journal

トレードジャーナル——心理の記録装置

Module 4でジャーナルの数値面（R, W, E, D）を解説しました。ここでは心理面の記録を追加します。

すべてのトレードについて、以下を記録する：

感
エントリー時の感情状態。「冷静」「焦り」「自信過剰」「不安」「リベンジ」など。1〜2語で記録するだけでいい。
遵
ルール遵守の自己採点。トレンドテンプレートを確認したか。損切りを設定したか。サイズは2%ルール以内か。Yes/Noで記録。
学
事後の教訓。「何を学んだか」を1文で。特に、ルールを破った場合はなぜ破ったかの原因分析。

20トレード後にジャーナルを振り返ると、自分の心理パターンが見えてきます。「焦り」の状態でエントリーしたトレードの勝率、「冷静」な状態でのそれ。データが自分のバイアスを教えてくれる。
