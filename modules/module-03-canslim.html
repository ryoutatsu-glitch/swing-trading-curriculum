Context

なぜCAN SLIMを学ぶのか

Module 1（Stage Analysis）で「いつ買えるか」を学び、Module 2（SEPA/VCP）で「どこで買うか」を学びました。CAN SLIMは「何を買うか」——銘柄の質をどう評価するかのフレームワークです。

O'Neilは1953年から2001年までの米国株のウィナー600銘柄を分析し、大幅上昇する前にこれらの銘柄が共通して持っていた特徴を7つの基準にまとめました。MinerviniはO'Neilの弟子筋にあたり、SEPAの「E（Earnings）」はCAN SLIMの体系から強い影響を受けています。

ゲーム全体の少なくとも50%は、相場全体の方向性だ。

— William O'Neil

この一文が示すように、O'Neilは個別銘柄のファンダメンタルだけでなく、市場全体の方向性（M）を最重要視しています。CAN SLIMの7文字のうち最後のMが最も大切だというのは、一見すると矛盾に見えますが、これが彼の核心的な教えです。

The Seven Criteria

CAN SLIMの7つの基準

各文字をクリックして詳細を展開してください。各基準の原則と、日本株で適用する際の注意点を解説します。

C

Current Quarterly Earnings

当期四半期のEPS（1株当たり利益）
+

原則

  • 直近四半期のEPSが前年同期比で最低25%以上増加していること
  • 40〜100%以上の成長であればさらに良い
  • 売上高も同時に前年同期比25%以上成長、または直近3四半期で加速していること
  • EPS成長率が2四半期連続で大幅に減速した銘柄は除外

なぜ重要か

  • 株価を動かすのは最終的に業績。足元の四半期決算が強い企業は、市場の関心と資金を集める
  • EPSだけでなく売上も見る理由は、コスト削減だけでEPSを伸ばしている企業を排除するため
🇯🇵 日本株での適用

日本企業の決算短信で四半期EPSを確認できるが、株式分割がEPSの遡及調整に反映されないケースがある。分割銘柄は表面上EPSが急落して見える。対策として、EPSの代わりに営業利益（前年同期比）で判定する方法が実用的。営業利益は株式分割の影響を受けない。

A

Annual Earnings Growth

年間EPS成長率
+

原則

  • 過去5年間で年間EPSが毎年増加していること
  • 年平均成長率（CAGR）25%以上が望ましい
  • 直近12ヶ月のEPSが直近通期のEPS以上であること（トレンドが続いている確認）
  • ROE（自己資本利益率）は17%以上が基準

CとAの関係

  • Cは「足元が強いか」、Aは「長期的に成長してきたか」。両方が揃って初めて候補になる
  • 一時的なEPSの急増（特別利益等）に騙されないために、Aの長期視点が必要
🇯🇵 日本株での適用

日本株の平均ROEは約10%で、米国（約18%）より低い。ROE基準を17%で適用すると候補が極端に少なくなる場合は、12〜15%に緩和して日本市場の実情に合わせるのも一案。ただしROEの水準を下げる場合、他の基準はより厳格に適用すること。

N

New Products, Management, Highs

「新しさ」——新製品・新経営陣・新高値
+

原則

  • 大きく上昇する銘柄には必ず何かの「新しさ」がある——新製品、新サービス、新経営陣、業界の構造変化
  • そして最も重要なのは、株価が新高値を更新しているタイミングで買うこと
  • 安いから買うのではなく、高いから買う。底値圏の株を拾うのはCAN SLIMの正反対

O'Neilの逆説

  • 「高すぎてリスクが高そうに見える銘柄ほど、さらに値上がりする」
  • 「安くて割安に見える銘柄ほど、さらに値下がりする」
  • これはModule 1のStage Analysisと完全に整合する。Stage 2（上昇中）で買え、Stage 4（下落中）は触るな
S

Supply and Demand

需給——発行済株式数と出来高
+

原則

  • 発行済株式数が少なめの銘柄のほうが、需要増加時に株価が大きく動く
  • 大型株より中小型株のほうが値動きが軽い。ただし流動性とのバランスは必要
  • 株価上昇時に出来高が増加し、下落時に出来高が減少するパターンを確認する
  • 直近2〜3年で負債比率が減少している企業は加点要素
🇯🇵 日本株での適用

63万円の資本で100株単位の取引をする場合、株価3,000円以下の銘柄が中心になる。中小型株が自然とフィルターされるため、S（需給）の条件は比較的満たしやすい。ただし出来高が極端に少ない銘柄は約定しにくいので、1日平均出来高5万株以上を最低ラインとする。

L

Leader or Laggard

業界のリーダーか、出遅れか
+

原則

  • 業界内の上位1〜3銘柄に集中する。「二番手」「出遅れ」は買わない
  • リーダーの定義は規模やブランド力ではなく、EPS成長率・売上成長率・株価のモメンタム（RS）が業界最強であること
  • RS（相対強度）が70以上、理想は80〜90台
  • 「リーダーが高すぎるから二番手を買う」は典型的な失敗パターン

Minerviniとの重複

  • Module 2のトレンドテンプレート8番目の条件「RS 70以上」はまさにこのLに対応
  • 両者とも「弱い銘柄を無視する」という同じ原則に立っている
I

Institutional Sponsorship

機関投資家の保有
+

原則

  • 最低3〜10の機関投資家が保有していること
  • 機関投資家がゼロの銘柄は流動性が低く、上昇力も弱い
  • ただし機関投資家の保有比率が高すぎるのも問題。悪材料で一斉に売られるリスクがある
  • 直近の四半期で機関投資家の保有数が増加傾向にあるかを確認する
🇯🇵 日本株での適用

日本株では四季報や有価証券報告書の「大株主の状況」で確認できる。信託口（信託銀行名義）や外国法人の保有があれば、機関投資家が入っている証拠。EDINETの大量保有報告書（5%ルール）も参考になる。中小型株では機関投資家が少ないのが普通なので、「ゼロでないこと」を最低条件とし、増加傾向を重視する。

M

Market Direction

市場全体の方向性——最も重要な要素
+

原則

  • O'Neilは「ゲーム全体の50%以上が市場全体の方向性」と断言している
  • どれだけ優れた銘柄でも、市場全体が下落基調なら逆らえない。3/4の銘柄は市場の方向に引きずられる
  • 市場が上昇トレンドのときだけ積極的に買い、下落トレンドでは現金化して待つ
  • 判定方法：主要指数（S&P 500 / 日経平均 / TOPIX）の日足チャートと出来高パターンを見る

「ディストリビューション・デイ」の概念

  • 主要指数が出来高の増加を伴って−0.2%以上下落した日を「ディストリビューション・デイ」と呼ぶ
  • 4〜5週間で4〜5日のディストリビューション・デイが発生したら、市場のトレンド転換を疑う
  • この時点で新規のエントリーを停止し、既存ポジションの利確を検討する

⚠ これが最も重要な教え：CAN SLIMの6文字（C, A, N, S, L, I）を完璧に満たしても、M（市場の方向性）が悪ければ買ってはいけない。地合いが悪いときに休めるかどうかが、生存率を決定的に左右する。

🇯🇵 日本株での適用

日経平均またはTOPIXの日足チャートで判定。200日移動平均線の上にあり、上向きなら上昇トレンド。加えて、ディストリビューション・デイのカウントを自分で記録する習慣をつける。これはトレードジャーナルに毎日1行追加するだけでできる。

Comparison

MinerviniとO'Neilの比較

両者の手法は多くの部分で重なります。違いは力点の置き方です。

MINERVINI (SEPA)

強み：エントリータイミングの精度。VCPによるタイトなストップ設定。リスク管理が厳格。

弱み：ファンダメンタル分析の体系はO'Neilほど詳細でない。

O'NEIL (CAN SLIM)

強み：ファンダメンタルの選定基準が体系的。市場全体の方向性判定。600銘柄のデータに裏付けされている。

弱み：エントリーポイントの定義がMinerviniほど精密でない。ストップルールも緩め。

組み合わせが最強。CAN SLIMでファンダメンタルをスクリーニングし、Minerviniのトレンドテンプレートで技術的に確認し、VCPでエントリーする。この3層フィルターが、個人投資家として取りうる最も堅実なアプローチのひとつです。

Workflow

実践ワークフロー：3モジュールの統合

Module 1〜3を統合すると、以下の手順で銘柄選定からエントリーまで進めます。

1
M（市場全体）を確認。日経平均/TOPIXが上昇トレンドか？ディストリビューション・デイが溜まっていないか？→ NOなら現金で待つ。
2
CAN SLIMでファンダを篩にかける。四半期EPS＋売上が25%以上成長、年間EPS増加傾向、RS上位、機関投資家保有あり。
3
Minerviniのトレンドテンプレートを適用。8項目すべて合格する銘柄だけ残す。ここで候補が5〜20銘柄に絞られる。
4
VCPを探す。残った候補のチャートを見て、ボラティリティの収縮パターンが形成されている銘柄を特定する。
5
ブレイクアウトでエントリー。ピボット突破＋出来高急増。ストップロスを設定。トレードジャーナルに記録。
