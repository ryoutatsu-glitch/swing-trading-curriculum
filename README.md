## 設計思想

- **巨人の肩に乗る**: 実績のある手法（Weinstein, Minervini, O'Neil）を土台にする
- **エッジの習得**: 汎用的なテクニカル分析スキルを実弾投入で体に叩き込む
- **損小利大**: リスク管理を手法と同格で学ぶ
